# Frigate Notify Bridge (Home Assistant Add-on)

Simple MQTT bridge for Frigate events.

## Features
- Subscribes to Frigate MQTT events
- Ready for Telegram / WhatsApp / Email extensions
- Home Assistant Add-on compatible

## Configuration
- mqtt_host: MQTT broker (default core-mosquitto)
- mqtt_port: 1883
- mqtt_topic: frigate/events
